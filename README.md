## gitを用いた共同開発のフロー

Gitを用いた共同開発のフローを、実際に手を動かしながら学べる講座です。

タイポや修正依頼はPRベースで出してもらえると助かります。

## setup

```
npm i -g vuepress
```

```
npm install
```

## build server

```
npm run dev 
```
